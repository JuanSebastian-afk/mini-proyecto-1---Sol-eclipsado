# Mini-proyecto N°1
Asignatura: Funtamentos de la programación orientada a eventos (FPOE).

Autores: Juan Sebastian Valencia, Andrés Leniz.

Este primer mini-proyecto de la asignatura de FPOE tiene como objetivo crear un juego
en el lenguaje de programación Java haciendo usa de la tecnologia JavaFX, Maven y SceneBuilder.

El juego consiste en adivinar una palabra letra por letra, se podra fallar un maximo de cinco ocaciones 
para hallar una palabra de entre seis y doce letras. Tambien se cuenta con la opción de ayuda para que el sistema
revele una letra de la palabra en su posición correspondiente, esta ayuda se podra utilizar un maximo de tres veces
por ronda.
